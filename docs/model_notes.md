# Model Notes

This document is for refining the trading model before live data or alert delivery is connected.

## Guardrails

- This app is an alert assistant only.
- It does not place trades.
- It does not connect to a broker.
- Any alert should be treated as research until validated.

## Concepts To Define

- Exact session windows and timezone.
- What counts as a valid NQ vs ES SMT divergence.
- Swing high and swing low sensitivity.
- How to anchor manipulation legs for standard deviation levels.
- OTE retracement range and acceptable entry zone.
- Momentum loss rules at deviation levels.
- Invalidation rules.
- Alert wording and priority levels.
